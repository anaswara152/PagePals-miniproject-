# PagePals – Library Management System

##  Project Overview

PagePals is a web-based Library Management System designed to manage books, users, and borrowing activities efficiently.
It helps automate library operations such as book issue, return, and catalog management.

---

##  Features

*  User Registration & Login (Students / Admin)
*  Add, Edit, Delete Books (Admin)
*  Search Books by Title / Author
*  Issue & Return Books
*  Track Due Dates
*  Admin Dashboard for Management

---

##  Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Python
* Django *(or Flask if you used Flask)*

**Database:**

* SQLite / MySQL

---

##  Installation & Setup

### 1️ Clone Repository

git clone https://github.com/your-username/pagepals.git

### 2️ Go to Project Folder

cd pagepals

### 3️ Create Virtual Environment

python -m venv venv

### 4️ Activate Environment

venv\Scripts\activate

### 5️ Install Dependencies

pip install -r requirements.txt

### 6️ Run Server

python manage.py runserver
